# Roteiro de Estudos PowerShell

## 1. Fundamentos
- Variáveis, tipos de dados, operadores
- Exercício: criar variáveis e operações matemáticas
- Mini-projeto: script que verifica se pessoa pode votar

## 2. Estruturas de Controle
- if, switch, loops
- Exercício: tabuada com loop
- Mini-projeto: script que gera tabuada

## 3. Pipeline e Cmdlets
- Uso de `Get-Process`, `Where-Object`, `Sort-Object`
- Exercício: filtrar processos por memória
- Mini-projeto: relatório de processos em CSV

## 4. Funções e Scripts
- Criar funções e scripts `.ps1`
- Exercício: função que soma dois números
- Mini-projeto: relatório de arquivos maiores que 1 MB

## 5. Tratamento de Erros
- try/catch/finally
- Exercício: abrir arquivo inexistente
- Mini-projeto: script de cópia com log de erros

## 6. Objetos e Customização
- `Get-Member`, `[PSCustomObject]`
- Exercício: criar objeto com Nome e Idade
- Mini-projeto: relatório de usuários

## 7. Automação de Tarefas
- Manipulação de arquivos, serviços, registro
- Exercício: copiar arquivos de uma pasta
- Mini-projeto: script de backup automático

## 8. Projetos Avançados
- Monitor de processos
- Gerador de relatórios de sistema
- Automação de usuários em AD
- Integração com APIs REST
